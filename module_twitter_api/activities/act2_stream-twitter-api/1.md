# Tweepy

**What is Tweepy?**

Tweepy is a Python library on accessing Twitter API. It is great for simple automation and creating twitter bots. It holds many features, such as getting tweets from your timeline, creating/deleting tweets, and following/unfollowing users.

Tweepy provides many methods and functions to access Twitter APIs. If you are curious and want to read more about them, Tweepy has a documentation page where all their methods are listed and explained. To view that, you can go to [https://tweepy.readthedocs.io/en/v3.5.0/](https://tweepy.readthedocs.io/en/v3.5.0/) to read more.
