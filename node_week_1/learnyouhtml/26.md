# FORMS

## THE CHALLENGE

Create a form with two inputs \(for login and password\) and a submit button.

Solution:

```markup
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>Links</title>
  </head>
  <body>
    <form action="#">
      <input type="text">
      <input type="password">
      <input type="submit">
    </form>
  </body>
</html>
```
