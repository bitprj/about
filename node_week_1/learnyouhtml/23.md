# LINKS

## THE CHALLENGE

In this exercise you have to create a clickable image \(link with image\). Use any image and any link you like.

Solution:

```markup
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>Links</title>
  </head>
  <body>
    <a href="#">
      <img src="path/to/img" alt="some image">
    </a>
  </body>
</html>
```
