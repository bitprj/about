# TABLES

## THE CHALLENGE

Create this table:

| Europe | Asia | Africa |
| :--- | :--- | :--- |
| Ukraine | China | Egypt |
| Poland | India | Kenya |
| Italy | Thailand | Sudan |

Hint: The first row is a heading row, so you should use the `<th>` tag.

Solution:

```markup
<table>
    <tr>
        <th>Europe</th>
        <th>Asia</th>
        <th>Africa</th>
    </tr>
    <tr>
        <td>Ukraine</td>
        <td>China</td>
        <td>Egypt</td>
    </tr>
    <tr>
        <td>Poland</td>
        <td>India</td>
        <td>Kenya</td>
    </tr>
    <tr>
        <td>Italy</td>
        <td>Thailand</td>
        <td>Sudan</td>
    </tr>
</table>
```
